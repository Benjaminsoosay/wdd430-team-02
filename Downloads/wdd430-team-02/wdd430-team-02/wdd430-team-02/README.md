## Project Purpose

Handcrafted Haven is a full-stack web application that connects artisans and crafters with customers who value unique, handcrafted items. The platform serves as a virtual marketplace where sellers can showcase their products, share their stories, and manage listings, while buyers can browse, filter, review, and purchase handmade goods. The project emphasises community, sustainable consumption, and support for local artisans.

## Design Theme

### Colors
- **Primary:** `#3B82F6` (blue) – buttons, links, headers
- **Secondary:** `#10B981` (green) – success states, accents
- **Background:** Light mode `#F9FAFB`, Dark mode `#111827`
- **Text:** Light mode `#1F2937`, Dark mode `#F9FAFB`

### Typography
- **Font Family:** Inter (sans-serif)
- **Headings:** Bold weight, responsive scale
- **Body:** Regular weight, line-height 1.5

### Layout & Components
- Centered container: `max-w-7xl mx-auto`
- Card-based grid for product listings
- Sticky header with navigation
- Soft shadows and hover effects

## User Stories (12 Core Features for MVP)

| # | User Story | Priority |
|---|------------|----------|
| 1 | As a new user, I want to create an account so I can save my profile and favorites. | High |
| 2 | As a returning user, I want to log in securely to access my dashboard. | High |
| 3 | As a seller, I want to add a new product with title, description, price, and images. | High |
| 4 | As a seller, I want to modify an existing product's details. | Medium |
| 5 | As a seller, I want to remove a product I no longer offer. | Medium |
| 6 | As a user, I want to see all products ordered by date or popularity. | High |
| 7 | As a seller, I want to mark an item as sold. | Medium |
| 8 | As a user, I want to filter products by category, price range, or availability. | Low |
| 9 | As a mobile user, I want the interface to work perfectly on any screen size. | High |
| 10 | As a user, I want to switch between dark and light themes. | Low |
| 11 | As a user, I want to search products by keyword. | Medium |
| 12 | As a user, I want my cart and preferences to persist after page refresh. | High |
