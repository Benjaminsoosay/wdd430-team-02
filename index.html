User Stories & Work Items

Minimum 10 required – showing all 12 · fully attributed to the new team

1. User Registration

As a new user, I want to create an account so I can save my tasks.

Priority: High

2. User Login

As a returning user, I want to log in securely to access my tasks.

Priority: High

3. Create Task

As a user, I want to add a new task with title, description, and due date.

Priority: High

4. Edit Task

As a user, I want to modify an existing task’s details.

Priority: Medium

5. Delete Task

As a user, I want to remove a task I no longer need.

Priority: Medium

6. View All Tasks

As a user, I want to see a list of all my tasks ordered by due date.

Priority: High

7. Mark Task Complete

As a user, I want to check off a task and see it visually distinguished.

Priority: Medium

8. Filter by Status

As a user, I want to filter tasks by All / Active / Completed.

Priority: Low

9. Responsive Layout

As a mobile user, I want the interface to work on any screen size.

Priority: High

10. Dark/Light Mode Toggle

As a user, I want a button to switch between dark and light themes.

Priority: Low

11. Search Tasks

As a user, I want to search tasks by keyword.

Priority: Medium

12. Persistent Storage

As a user, I want my tasks to remain after page refresh (localStorage or DB).

Priority: High
